# API Models, Serializers & Endpoints

The server-side data model, DRF serializers and API views that expose Isabl resources and result files.

- [Views & Endpoints](./views-endpoints/) — API views to read, stream and download analysis results, manage submissions, registration and user preferences.
- [Routing & URL Patterns](./routing-url-patterns/) — Top-level API URL patterns and the automatic model route generation helpers.
