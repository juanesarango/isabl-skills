# Validation Rules & Sample/Assay Checks

A comprehensive set of validations that application authors and operators use to enforce input, sample pairing, assay, platform and species constraints.

## Contents

- [Input & File Validations](./input-file-validations/) — Checks that required files, reference indexes and raw inputs exist and match expected formats.
- [Sample Pairing & Roles](./sample-pairing-roles/) — Validators that enforce correct target/reference pairing, single-target constraints and normal/control assignments.
- [Assay, Platform & Species Checks](./assay-platform-species-checks/) — Validators for experimental method compatibility, platform matching, species constraints and source material checks (PDX, DNA/RNA).
