# Applications, Analyses & Merging

Develop, validate and run metadata-driven applications; collect results; and merge/aggregate analyses at individual or project scope.

## Contents

- [Developing Applications](./developing-applications/) — APIs and hooks application authors implement: lifecycle, result collection and CLI metadata helpers.
- [Running Analyses & Submission](./running-analyses-submission/) — Orchestration and submission paths for running analyses, including summary reporting and submission controls.
- [Merging & Aggregation](./merging-aggregation/) — Hooks and commands to merge multiple analyses into aggregated project- or individual-level analyses and to trigger merge jobs.
- [Validation Rules & Sample/Assay Checks](./validation-rules-sampleassay-checks/) — A comprehensive set of validations that application authors and operators use to enforce input, sample pairing, assay, platform and species constraints.
