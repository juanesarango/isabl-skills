# Tutorials, Tips & Community

Guides for contributors, practical tips and pointers to common CLI workflows and UX improvements.

## Source Documents

- **Tips and Tricks for Isabl CLI** — Collects helpful extra features and UX tips to improve the Isabl experience, including enhancements like colored logs when using batch systems. The page highlights small customizations and practical tips that make using Isabl-CLI and job submission more efficient and readable.
  [Source](https://docs.isabl.io/bonus-tips)
- **Contributing Guide: Development Environment Setup** — Provides a step-by-step tutorial to set up a full development environment for Isabl components and explains how to contribute. It covers cloning the repositories, local development practices, and encourages community contributions to code, documentation, and tests.
  [Source](https://docs.isabl.io/contributing-guide)
- **Isabl CLI: Commands, Retrieval, and Management** — Summarizes the major capabilities of the isabl-cli package, including creating/executing analyses, retrieving data, and importing/managing files. The page highlights that the CLI exposes many useful functions for common workflows and points users to specific command areas for analyses, data retrieval, and imports.
  [Source](https://docs.isabl.io/other-cli-commands)
