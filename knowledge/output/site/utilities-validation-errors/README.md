# Utilities, Validation & Errors

General-purpose utilities, filesystem helpers, validators and the library's custom exception types for robust error handling.

## Contents

- [Exceptions & Error Types](./exceptions-error-types/) — Custom exception hierarchy used across the CLI and import machinery to signal validation, configuration and automation errors.
- [Filesystem & Transfer Utilities](./filesystem-transfer-utilities/) — Helpers for directory creation, transfers, archive creation, linking and ownership/permission checks used by import and storage workflows.
- [CLI Helpers & Misc Utilities](./cli-helpers-misc-utilities/) — General helpers for CLI UX, traversal, decorator composition, analytics wrappers, searching files, and small debugging utilities.
- [Validators & CLI Callbacks](./validators-cli-callbacks/) — Small validation utilities used for CLI argument validation: directory/file pattern checks and parsing sample pair TSVs.
