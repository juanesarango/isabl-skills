# Overview & Getting Started

Home page, quick start and onboarding materials to get an Isabl instance running and to register metadata/import data.

## Source Documents

- **Isabl documentation homepage and platform overview** — The Isabl Docs home page provides an overview of the Isabl platform, its features, and links to core documentation sections such as Quick Start, Data Model, Importing and Retrieving Data, Applications, Deployment and Maintenance. It also includes links to community resources and a brief description of the platform's goals and origin.
  [Source](https://docs.isabl.io/)
- **10-Minute Quick Start Guide to Isabl Platform** — A short tutorial that walks new users through installing Isabl, registering metadata, importing data, and running automated processing workflows — designed to get a working instance up and running in about ten minutes. It also covers licensing (research/academic vs commercial) and points to support channels and additional guides.
  [Source](https://docs.isabl.io/quick-start)
