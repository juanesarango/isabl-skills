# Utilities & Filesystem Helpers

General-purpose utilities used across the CLI and importers for filesystem operations, archiving, rsync command generation, results discovery, UX messages and debugging aids.

## Contents

- [Filesystem, Links & Archiving](./filesystem-links-archiving/) — Directory creation honoring mode, forced link/symlink helpers, tar archiving and directory size inspection utilities.
- [Transfers & rsync Helpers](./transfers-rsync-helpers/) — Helpers to build rsync command lines and to check installed rsync compatibility for append/verify features.
- [Results Discovery & Experiment Utilities](./results-discovery-experiment-utilities/) — Utilities that find analysis results on experiment records, wrapper helpers to obtain single results with clearer errors, and traversal helpers for nested dicts.
- [CLI UX, Analytics & Debug Helpers](./cli-ux-analytics-debug-helpers/) — Message helpers for commit guidance and titles, decorators/application wrappers to send telemetry, and debugging utilities.
