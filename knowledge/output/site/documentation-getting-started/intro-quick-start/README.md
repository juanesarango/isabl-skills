# Intro & Quick Start

Landing pages and short tutorials to get started with Isabl and community guidance.

## Source Documents

- **Isabl documentation home and feature overview** — The main documentation landing page for Isabl, introducing the platform and linking to key sections such as Quick Start, data model, importing/retrieving data, application development, deployment, and contributing. It outlines Isabl's purpose as a plug-and-play framework for managing patient-centric multimodal data and points to further resources.
  [Source](https://docs.isabl.io/)
- **Quick Start: Ten-minute Isabl Tutorial** — A concise 10-minute tutorial that walks new users through installing Isabl, registering metadata, importing data, and running automated processing. It also summarizes licensing (free for research/academic use), where to find source code, and community support channels.
  [Source](https://docs.isabl.io/quick-start)
- **Bonus Tips and Enhancements for Isabl CLI** — Provides extra features and usability tips to improve the Isabl experience, with practical examples and visual aids. The document highlights enhancements like colored logs with batch systems and other CLI conveniences that make debugging and monitoring jobs easier. It is aimed at power users who want to maximize productivity and visibility when running Isabl workflows.
  [Source](https://docs.isabl.io/bonus-tips)
- **Contributing and Setting Up Development Environment** — Step-by-step contributor tutorial to set up a full local development environment for the Isabl stack. It covers cloning the isabl_api repository and other components, guidance on where to add documentation or code, and encourages community contributions with crediting and coding principles. The guide is aimed at developers who want to extend Isabl or run it locally for development and testing.
  [Source](https://docs.isabl.io/contributing-guide)
- **Isabl CLI: Additional Useful Commands Overview** — Summarizes a set of built-in isabl-cli commands that support three main activities: creating/executing analyses, retrieving information, and importing/managing workspace files. The page links to detailed sections on running applications, retrieving data, and importing data and gives examples for getting started with isabl command-line utilities. It helps users to discover CLI features that extend automation and data management workflows.
  [Source](https://docs.isabl.io/other-cli-commands)
